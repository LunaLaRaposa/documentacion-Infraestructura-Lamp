# Necesidades:
- Un servidor web Apache con PHP (LAMP) en una distribución Linux (ej. Ubuntu Server 22.04).
- Un servidor de base de datos MySQL/MariaDB con una base de datos para la web y otra para la gestión - interna.
- Acceso remoto seguro (SSH) y configuración de firewall básico (UFW).
- Monitorización del sistema con una herramienta sencilla (ej. Netdata o script personalizado de check).
- Estrategia de copias de seguridad automáticas (mysqldump + rsync) con rotación.
- Documentación para el cliente: guía de mantenimiento y plan de recuperación ante desastres.
